# Healthy Progress Visualization Skill

## Quando usar

Use ao criar ou revisar indicadores visuais de progresso, Placar, Jardim, retomadas, streaks, badges, alertas de cuidado ou estados de baixa execucao.

## Instrucoes praticas

1. Progresso pequeno deve aparecer.
2. Retomada deve ser celebrada como retorno ao caminho.
3. Baixa execucao deve virar ajuste de friccao.
4. Nao usar vergonha, culpa espiritual, ranking publico ou derrota visual.
5. Evitar vermelho agressivo para cuidado comum.
6. Indicadores devem responder: o que avancou, o que precisa de cuidado e qual e o proximo passo.
7. Quando houver area negligenciada, manter dignidade visual e convite concreto.
