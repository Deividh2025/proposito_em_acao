# Restart Mode Skill

## Quando usar

Use quando a experiencia envolve atraso, falha, abandono, tarefa parada, habito quebrado, Placar, Jardim, Revisao Semanal ou retomada.

## Instrucoes praticas

1. Valorize retorno acima de perfeicao.
2. Pergunte pelo menor retorno honesto e pelo ajuste de rota.
3. Evite vermelho agressivo, ranking, streak quebrado como derrota ou linguagem de fracasso.
4. Mostre responsabilidade sem humilhacao.
5. Converta queda em aprendizado, microacao ou descanso legitimo.
6. No Placar e Jardim, sinalize cuidado necessario como convite, nao como punicao.
7. Se houver Atalaia, nenhuma mensagem de falha deve sair sem previa e consentimento.
