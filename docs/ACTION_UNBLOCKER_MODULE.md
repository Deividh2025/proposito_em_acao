# Action Unblocker Module

## Objetivo

O Desbloqueador de Acao transforma travas operacionais em primeiro passo pequeno, versao minima, foco recomendado e rota segura.

## Fluxo

1. Usuario informa tarefa, energia, tempo disponivel, obstaculo e tom.
2. Mock/IA estruturada sugere primeiro passo, microtarefas, versao minima e rota.
3. Bloqueio operacional pode ir para Foco.
4. Bloqueio cognitivo/emocional pode sugerir Metacognicao.
5. Crise ou risco grave sai do fluxo de produtividade e orienta ajuda humana adequada.

## Integracao com Prompt 11

Prompt 11 usa o Desbloqueador como rota manual quando o usuario trava durante foco. `action_unblock_session_id` esta preparado em `focus_sessions` para rastrear origem sem expor dados ao Atalaia.

## Privacidade

Sessoes sao owner-only. Nada e enviado ao Atalaia automaticamente. Conteudo bruto nao deve ser logado.

## Fora de escopo

Diagnostico, terapia, promessa de cura, afirmacao de vontade divina especifica, Atalaia funcional e OpenAI real acionada por UI sem autorizacao.
