# Knowledge Base

Base interna futura para materiais aprovados que poderao apoiar a IA do Proposito em Acao.

## Uso

Esta pasta ainda contem apenas placeholders. Nenhum material real foi carregado, indexado ou enviado a vector store/file search.

## Versionamento

Materiais futuros devem registrar fonte, autor, data, versao, permissao de uso e aprovacao do fundador antes de qualquer uso por IA.

## Limites

Nao adicionar conteudo protegido, clinico, pastoral, teologico, cientifico ou pessoal sem autorizacao explicita. A IA deve usar estes materiais como contexto auxiliar, nao como autoridade absoluta.
