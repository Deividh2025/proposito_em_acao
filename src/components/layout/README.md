# Layout Components

Componentes reutilizaveis do shell do App Router.

## Componentes atuais

- `AppShell`: composicao raiz visual.
- `Sidebar`: navegacao desktop.
- `Topbar`: contexto e acao primaria no desktop.
- `MainContent`: area principal responsiva.
- `RightPanel`: modos de apoio e privacidade no desktop.
- `MobileShell`: navegacao curta para PWA/mobile complementar.
- `PageHeader`: cabecalho de pagina.
- `SectionHeader`: cabecalho de secao.
- `PlaceholderPage`: contrato visual de paginas em preparacao.

Esses componentes sao fundacionais. Eles nao implementam fluxos finais de produto.
