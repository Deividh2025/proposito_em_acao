# UI Components

Primitivos visuais reutilizaveis do design system inicial.

## Componentes atuais

- `Button`, `IconButton`
- `Card`, `Panel`
- `Badge`, `Tag`
- `Input`, `Textarea`, `Select`
- `Checkbox`, `RadioGroup`, `Switch`, `Slider`
- `Progress`, `Stepper`, `Tabs`
- `Modal`, `Drawer`, `Tooltip`, `Toast`
- `EmptyState`, `LoadingState`, `ErrorState`, `SuccessState`

## Limites

Os componentes sao base visual. Modais, drawers, tabs e formularios interativos completos devem receber comportamento acessivel real quando forem usados em fluxo final.
